Información sobre la API

El sueño del futbolero es un API que te permitirá conocer información sobre cualquier camiseta de futbol de cualquier club. Cada producto tendrá información tanto de la camiseta como del club de la misma. Cada elemento tendrá sus propios endpoints a través del cual te permitirá acceder a la información de cada elemento

Camisetas

  • getProductosDisponibles() Permite visualizar lista completa de camisetas disponibles.
  • getProductosNoDisponibles() Permite visualizar lista completa de camisetas no disponibles.
  • getProductosMayorPrecio() Permite ordenar las camisetas segun su precio de mayor a menor
  • getProductosMenorrPrecio() Permite ordenar las camisetas segun su precio de menor a mayor
  • searchId() Permite buscar por id en el listado de camisetas.
  • searchPais() Permite buscar por país en el listado de camisetas.
  • searchMarca() Permite buscar por marca en el listado de camisetas.
  • createProducto() Permite crear una nueva camiseta.
  • updateProducto() Permite actualizar los datos de una camiseta ya existente.
  • desactivateProducto() Permite quitar del listado de camisetas disponibles pero no las elimina. Este endpoint se piensa desde el lado de que el prodcuto pueda volver a ingresar.

Clubes

  • getClub() Permite visualizar lista completa de clubes.
  • createClub() Permite crear un nuevo club.
  • updateClub() Permite actualizar los datos de un club ya existente.

Usuarios

  • getUsuario() Permite visualizar lista completa de usuarios.
  • createUsuario() Permite crear un nuevo usuario.
  • updateClub() Permite actualizar los datos de un usuario ya existente.
